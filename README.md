# VideoFX Scanner

Реалтайм-обработчик видео в браузере с эстетикой камер видеонаблюдения.
Источник — видеофайл / камера телефона (WebRTC) / Kinect. Поверх — слои эффектов:

- **2D · эффекты (по умолчанию):** тепловизор/ИК, обводка объектов (Sobel),
  пиксель-арт/ASCII по глубине, маски людей (YOLOv8-seg / Selfie), детекция движения
  боксами, лица (сетка 468 точек), руки + жесты, constellation-сетка. Всё на GPU-шейдере
  + воркеры/детекторы вне рендер-кадра. **База работает в любом браузере на 60 fps, сразу.**
- **3D · глубина:** облако точек из нейро-карты глубины (Depth Anything V2, WebGPU)
  + эффект-сканер по Z. Опционально, требует WebGPU. Режим `composite` накладывает 2D снизу.

Слои с модификаторами и драйверами параметров — это уже дата-флоу; целевая архитектура
и дорожная карта в [ARCHITECTURE.md](ARCHITECTURE.md), конвенции кода в [CLAUDE.md](CLAUDE.md).

## Стек

React 19 · @react-three/fiber 9 · three.js · @huggingface/transformers (WebGPU, только 3D-режим)

## Запуск

```bash
npm install
npm run dev
```

Открой адрес из консоли (обычно http://localhost:5173) и загрузи видеофайл в панели
справа. Базовые 2D-эффекты работают в любом современном браузере. Для 3D-режима
(глубина) нужен **WebGPU** (свежие Chrome/Edge; в Safari — включить в экспериментальных
функциях); первое включение скачает веса модели (~50 МБ).

## Чего ожидать по скорости

- **2D-эффекты (шейдер):** 60 fps, чистый GPU.
- **Движение + точки:** vision-воркер на кадре 192px — десятки fps в отдельном потоке,
  на рендер не влияет (рендер переиспользует последний результат).
- **Глубина (3D-режим):** на нормальной GPU через WebGPU — 15–30 инференсов/с на входе
  384px; считается не каждый кадр (ползунок «инференс: каждый N-й кадр»).

Если тормозит: в 3D-режиме увеличь «каждый N-й кадр»; в 2D — уменьши «кол-во точек».

## Важно (честные оговорки)

- 2D-режим — основная **рабочая база**: собирается и работает без WebGPU и нейронок.
- 3D-режим: `onnx-community/depth-anything-v2-small` даёт относительную глубину
  (не метры) — для визуальных эффектов достаточно. Модель качается с HF hub при
  первом включении режима.
- «Сканирование» здесь = эффект поверх кадра/мгновенной глубины, без накопления единой
  3D-модели (это был бы SLAM — отдельная большая задача).
