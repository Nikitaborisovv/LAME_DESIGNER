# 2D-эффекты экранного пространства

> Эта папка — историческая точка расширения. Flat-режим уже реализован (не через
> `postprocessing`, а одним полноэкранным quad'ом). Реальный путь добавления эффектов —
> ниже; целевая модель (Map2D-операторы) — в [ARCHITECTURE.md](../../../ARCHITECTURE.md) §2.1.

## Как добавить 2D-эффект сейчас

Flat-режим — это один GPU-шейдер на полноэкранном quad'е, а не цепочка эффектов:

- GLSL — `src/render/flatShader.ts` (тепловизор, Sobel, пиксель-арт/ASCII, маски людей).
- Проброс uniform'ов из конфига — `src/render/FlatView.tsx` (`useFrame`).
- Параметры — поле в `SceneConfig` (`src/core/types.ts`) + спека слоя в
  `src/core/layerRegistry.ts` + контрол в UI.

Новый эффект-маска (глитч, голограмма, night-vision, edge-only «призрак») добавляется
веткой в `flatShader.ts` + uniform'ы + поле конфига + контрол. Без скрытого состояния —
только через `SceneConfig` (см. конвенции в [CLAUDE.md](../../../CLAUDE.md)).

## Куда это движется

В целевой архитектуре каждый такой эффект — оператор семейства **Map2D** (TOP-подобный):
`Level`, `Threshold`, `Composite`, `Mask`, `Ramp`, `Displace` и т.д. поверх общей
FBO-инфраструктуры (та, что уже используется для bloom во `FlatView`). Тогда эффекты
композируются как узлы, а маски (включая отрендеренные точки через `Splat`) работают как
альфа/маска для любого входа. Детали — [ARCHITECTURE.md](../../../ARCHITECTURE.md).
